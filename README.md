# VHDL Processor Basics

This repository contains essential hardware modules developed for the **Processor Design** course.

## Purpose
To build a library of fundamental digital components using a hierarchical VHDL approach, providing the building blocks for a future processor architecture.

## Core Components
- **Logic Gates:** NOT, AND, OR, XOR.
- **Multiplexers:** 2x1 and 4-bit bus versions.
- **Arithmetic:** 1-bit Full Adder and 4-bit Ripple Carry Adder.
- **Decoders:** 3x8 Binary Decoder.

## Project Structure
- `Entities/`: VHDL source files for each component.
- `components_pkg.vhd`: Central package for component declarations and easy reuse.
